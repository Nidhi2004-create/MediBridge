# MediBridge
MediBridge is a smart healthcare platform that converts unclear handwritten prescriptions into structured digital records, helps patients manage their treatment effectively, and enables caretakers to monitor health remotely.

Features:
 Prescription Digitization
Converts handwritten prescriptions into readable digital format
Reduces errors caused by unclear doctor handwriting
 Treatment Management
Track medications, dosage, and schedules
Set reminders for medicines
 Caretaker Monitoring
Allows family members or caretakers to monitor patient health remotely
Real-time updates on medication and health status
Health Records Storage
Securely stores patient history and prescriptions
Easy access anytime, anywhere
 User-Friendly Interface
Simple and intuitive UI for all age groups


Tech Stack
Frontend: React.js / HTML / CSS / JavaScript
Backend: Node.js / Express.js
Database: MongoDB
Other Tools: Git, GitHub, APIs
